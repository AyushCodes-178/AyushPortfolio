# Personal Portfolio Website

A personal portfolio website built from scratch using HTML and CSS.

## Features

- Home Page
  - Introduction and personal profile
  - Skills and interests
  - Contact section
  - Custom UI design

- Projects Page
  - **Unreal Engine game development** projects
  - Canva design projects (Posters & PPTs)
  - Project previews and descriptions

## Technologies Used

- HTML5
- CSS3

## Learning Journey

This project was created from scratch as part of my web development learning journey. It helped me practice:
- HTML page structure
- CSS styling
- Layout design
- Responsive web design
- Portfolio organization
